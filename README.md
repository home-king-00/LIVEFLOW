# LIVEFLOW

### 온라인 이벤트 솔루션 LIVEFLOW

> LIVEFLOW는 온라인 이벤트 솔루션 홈페이지입니다.  
> 이 프로젝트는 퍼블리싱을 담당하여 제작하였습니다.  
> 개발 환경은 http-server로 서버를 구성하여 개발하였습니다.

---

- LIVEFLO(약 15 페이지)
    - 프론트엔드
        - Jquery
        - Bootstrap
    - 협업 프로그램
        - Zeplin
